# Claude Code Shortcuts and Slash Commands

## Built-in Slash Commands (Skills)
- `/help` -- get help with Claude Code
- `/init` -- initialize a new CLAUDE.md file
- `/review` -- review a pull request
- `/security-review` -- security review of pending changes
- `/simplify` -- review changed code for quality and efficiency
- `/fast` -- toggle fast mode (Opus 4.6 with faster output)
- `/update-config` -- configure settings.json, hooks, permissions
- `/keybindings-help` -- customize keyboard shortcuts
- `/session-start-hook` -- set up SessionStart hook for web sessions
- `/loop` -- run a command on a recurring interval
- `/claude-api` -- help with Claude API / Anthropic SDK
- `/fewer-permission-prompts` -- add allowlist to reduce prompts
