# AI Advisor -- Kristin Baillie

This repo is Kristin's external memory for AI knowledge. It captures tools, prompts, workflows, Claude Code features, lessons learned, and terminology.

## Structure

- `tools/` -- AI tools Kristin uses or has evaluated
- `prompts/` -- Effective prompts with context on when to use them
- `workflows/` -- Multi-step processes combining AI tools
- `claude-code/` -- Claude Code features, hooks, MCP servers, agents, settings
- `lessons/` -- What didn't work and why
- `terminology/` -- AI concepts in plain language
- `CLAUDE.md` -- Agent instructions

## Rules

- No em dashes. No emojis unless requested.
- Short and direct.
- Never store API keys, passwords, or personal identifying information (repo is public).
