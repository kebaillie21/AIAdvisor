# CoWork

## Status: Active

## What It Is
A project management workspace with AI built in. Kristin uses it for professional projects: YPE Curriculum Edits, Attendance Data, Discipline Data Analysis, SEBC Conference, Conference Poster, Navigating Risky Waters, Organize Me, Copyediting.

## Key Features Seen
- Projects view with recent tasks
- Scheduled tasks
- Customize options (possible shortcut/template features -- not yet explored)
- Dispatch feature
- Recent conversations per project

## Does NOT Communicate with Claude Code
CoWork and Claude Code are separate tools. Slash commands created in Claude Code do not work in CoWork. Custom shortcuts for CoWork must be set up within CoWork's own features (check Customize in sidebar).

## Context Continuity in CoWork
No built-in shortcut yet discovered. Current workaround: paste a context-continuity prompt from `prompts/context-continuity.md` manually at the end of a CoWork chat before starting a new one.
