# Pocket AI

## Status: Active

## Purpose
Conversation capture and retrieval. Kristin uses it for meeting notes and searchable conversation memory.

## MCP Integration
Pocket AI has an MCP server that connects to Claude Code. Available tools:
- `search_pocket_conversations` -- search past conversations by keyword
- `search_pocket_conversations_timerange` -- search by date range
- `search_pocket_actionitems` -- find action items across conversations
- `get_pocket_conversation` -- retrieve a specific conversation
