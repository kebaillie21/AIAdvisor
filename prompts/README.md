# Prompts

Effective prompts Kristin has discovered, with context on when to use them.

## Format for Each Prompt Entry

```
### [Short Name]
**Context:** When to use this prompt
**Tool:** Which AI tool to use it with
**Prompt:**
[exact prompt text]
**Notes:** What makes it work, variations, caveats
```

## Categories
- `capture.md` -- prompts for capturing and organizing information
- `analysis.md` -- prompts for analyzing data or documents
- `writing.md` -- prompts for writing tasks
- `coding.md` -- prompts for coding and technical tasks
- `research.md` -- prompts for research tasks

Add new prompts to the appropriate category file. If no category fits, add here.
